Invoice vs Quote: what’s the difference?

A quote is a price proposal. An invoice is a request for payment. Use the right document at the right time.

Quick comparison

DocumentPurposeWhen to send
QuotePropose price + scopeBefore work starts
InvoiceRequest paymentAfter work delivered (or milestone)

Common mistakes

  • Sending an invoice before the client agrees on scope.
  • Leaving out expiry/valid‑until on quotes.
  • Using the same number format for both (quotes should have their own prefix).
